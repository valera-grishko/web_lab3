## Install requirements
```pip install -r requirements.txt```
## Run server
```python manage.py runserver```
## Run with websockets
```daphne blog.asgi:application```
## Create superuser
```python manage.py createsuperuser```
## Create migrations
```python manage.py makemigrations```
## Migrate
```python manage.py migrate```
